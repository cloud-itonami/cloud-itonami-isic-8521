About this blueprint

An open-source business blueprint for secondary education operators. This repository publishes a production-ready actor that any licensed secondary-school operator can fork, deploy, run, improve and sell — handling student intake through grading and graduation finalization with built-in compliance safeguards.

Built on a battle-tested state graph runtime with two independent layers of governance: a sealed SchoolOps-LLM advisor and an independent Curriculum Safeguarding Governor that enforces spec-backed jurisdiction requirements, complete evidence review, sufficient attendance hours, academic-integrity screening, and graduation-requirement verification.

Finalizing a real grading or graduation is never autonomous — both the Governor and the phase table enforce human sign-off at all times. Every intake, assessment, screening, grading and graduation path is immutably audited.
